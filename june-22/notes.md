# June 22, 2026 - HTML Tags and Multimedia

## What I Studied Today
Today was the foundation of web development - HTML!

### Topics Covered:
- HTML document structure (<!DOCTYPE>, <html>, <head>, <body>)
- Commonly used HTML tags like headings (h1-h6), paragraphs, lists
- Multimedia tags: <img> to embed images
- <video> and <audio> tags to add media to webpages
- Attributes like src, alt, width, height, controls
- How browsers read and render HTML structure

HTML is the skeleton of every webpage. Multimedia makes pages rich and interactive.
